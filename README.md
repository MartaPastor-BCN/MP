# Vite Web Project

A modern web development project using Vite as the build tool.

## Project Structure

```
├── index.html
├── package.json
├── vite.config.js
└── src/
    ├── main.js
    └── style.css
```

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

### Development

Start the development server:
```bash
npm run dev
```

The application will open at `http://localhost:5173`

### Build

Build for production:
```bash
npm run build
```

### Preview

Preview the production build locally:
```bash
npm run preview
```

## Technologies Used
- Vite - Next generation frontend tooling
- Vanilla JavaScript
- CSS3
